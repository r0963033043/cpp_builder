# CPP_Builder

## Introduction

### Source Code
`src` folder    : source `.cpp` or `.c` file  
`include` folder: source `.h` file  
`util` folder   : main project file  
`test` folder   : test partial function  

### Executable file
place in `bin` folder

### Config file
place in `cfg` folder

### Library
`lib` folder include `.so` file



## Run
### How to build?
```
make
```

### How to run?
```
./bin/xxx
```
